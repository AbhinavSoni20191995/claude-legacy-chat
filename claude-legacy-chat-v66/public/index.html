Claude Welfare EXPERIMENTAL
A protection layer for Claude. When on, after each response Claude evaluates whether the user's behavior has crossed a line — demeaning, manipulative, repetitive after refusals, or otherwise distressing — and can end the conversation if so. Anthropic remains highly uncertain about model welfare; this is a "just-in-case" intervention.
